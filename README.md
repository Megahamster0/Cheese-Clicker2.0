Cheese Clicker ist ein lustiges Clicker-Spiel, bei dem du so viel Käse wie möglich sammeln musst!  

## 🎮 Features  
- Klicke auf den Käse, um Punkte zu sammeln  
- Kaufe Upgrades, um mehr Käse zu bekommen  🧀
- Freischalte geheime Items
